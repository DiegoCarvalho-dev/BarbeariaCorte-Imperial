# La Mafia Barbearia Clone

This repository contains a pixel-perfect recreation of the La Mafia Barbearia homepage.

## Technologies

- React 19 with TypeScript
- Vite build tool
- Tailwind CSS for styling
- lucide-react for icons

## Setup

```bash
npm install
npm run dev
```

The development server will open automatically in your browser.

## Structure

- `src/components` – individual sections (Header, Hero, Services, etc.)
- `src/components/ui` – shared UI elements like `Button`
- `src/types.ts` – shared TypeScript interfaces

## Customization

Feel free to adjust colors in `tailwind.config.js`, swap images and text, or extend components with props.

## Notes

This project uses TypeScript strict mode for safety. Tailwind utilities provide responsive layout and custom colors matching the original design.
